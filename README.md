# Retail Customer Behavior Analysis

## Project Overview
This project provides a comprehensive analysis of retail customer behavior to identify key shopping trends and patterns. By utilizing an end-to-end data pipeline, this project demonstrates the ability to transform raw data into actionable business insights.

## Workflow & Methodology
The project follows a structured three-stage data analysis pipeline:

1.  **Data Cleaning (Python/Pandas):** Handled missing values, standardized data formats, and performed column manipulations to ensure data quality and consistency.
2.  **Data Management (SQL):** Imported the cleaned dataset into a database and performed complex queries to extract meaningful insights and aggregate metrics.
3.  **Visualization (Power BI):** Developed an interactive dashboard to visualize key performance indicators (KPIs), enabling a clearer understanding of customer segments, revenue, and behavior.

## Key Insights
* **Customer Segmentation:** Identified key patterns in behavior based on subscription status and demographic categories.
* **Revenue Performance:** Analyzed revenue distribution across different product categories.
* **Actionable Metrics:** Created clear visualizations for average purchase amounts and customer review ratings to support data-driven decision-making.

## Tech Stack
* **Programming:** Python (Pandas)
* **Database:** SQL
* **Visualization:** Power BI
